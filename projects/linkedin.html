LinkedIn data analysis

In this project I delve into the data job market by analyzing recent past trends(2024 late January).

The result final of this work is a Tableau dashboard, that you can interact with on Tableau public.

Data preparation and transformation

To prepare raw data for analysis, I used PySpark. For preparation, I checked, that the datasets were properly read, and the infered schemas were accurate. Secondly there were columns, that are essentially booleans encoded with strings 't' and 'n', so I had to convert them into boolean. Job skills were stored in a comma-delimeted string, so I split the strings, and obtained an ArrayType column. The columns, that will be used for joining were link columns, links are long strings, so for performance considerations, I hashed them using xxhash64.
